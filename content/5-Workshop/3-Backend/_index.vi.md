## Xây dựng Backend
Trong workshop này, bạn sẽ:

- Sử dụng **Amazon S3** để lưu trữ file dữ liệu đầu vào (CSV).  
- Cấu hình **AWS Lambda Trigger** để tự động import dữ liệu vào **DynamoDB**.  
- Tạo **Lambda Function (API Handler)** và công khai qua **API Gateway** để truy cập dữ liệu.  
- Kiểm thử API bằng **Postman** hoặc **API Gateway Console**.  
- Dọn dẹp toàn bộ tài nguyên sau khi hoàn thành.
